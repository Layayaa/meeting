# 会议室预约小程序（CloudBase）

基于微信小程序 + 云开发（CloudBase）的会议室预约系统，包含用户端预约流程与管理员后台管理功能。

## 功能概览

- 用户登录/注册：手机号 + 昵称 + 密码
- 预约会议室：支持未来 7 天预约，时段范围 `08:00-22:00`（按 1 小时）
- 我的预约：查看记录、按规则取消预约
- 活动列表：查看活动信息与二维码
- 管理后台：
  - 用户管理（添加用户、启用/禁用、派发次数）
  - 预约管理（筛选、强制取消）
  - 活动管理（发布、编辑、删除）
  - 次数规则设置（默认每周次数、手动刷新）

## 业务规则（核心）

- 权限分层：普通用户 / 已授权用户 / 管理员
- 次数计算：`剩余 = 默认次数 + 额外派发 - 已使用`
- 取消规则：会议开始前 2 小时可取消并返还次数
- 冲突规则：同会议室同日期同时间不可重复预约
- 连续时段限制：同账号不可预约连续时段

## 项目结构

- `miniprogram/`：小程序前端页面与资源
- `cloudfunctions/`：云函数（业务逻辑）
  - `registerUser`、`bindPhone`、`getUserStatus`
  - `reserveRoom`、`getReservationsByDate`
  - `serviceFunctions`（管理与聚合能力）
  - `weeklyReset`（每周次数刷新）

## 部署说明

1. 使用微信开发者工具打开项目根目录。
2. 在云函数目录中上传并部署所需函数（建议“云端安装依赖”）。
3. 初始化数据库集合：`users`、`admins`、`reservations`、`activities`、`settings`、`operation_logs`。
4. 配置管理员手机号到 `admins` 集合。
5. 配置 `weeklyReset` 定时触发（每周六 22:00）。

## 技术栈

- 微信小程序（WXML / WXSS / JS）
- CloudBase 云数据库
- CloudBase 云函数（Node.js）
